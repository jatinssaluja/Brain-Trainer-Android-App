          ******************************* Brain Trainer Android App *************************************************


This app presents a series of sums that the user can answer within 30 seconds. As soon as the user answers the sum of two numbers, the updated score is displayed on the screen and the user is notified whether the answer is correct or wrong. When the timer runs out, all buttons are disabled, a summary of score is displayed and the user is presented with "Play Again" button to start the quiz anew.
